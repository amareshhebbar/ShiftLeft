# ShiftLeft — Screenshots Guide



## Tools

- **Capture:** macOS `Cmd+Shift+4` / Linux `Flameshot` / Windows `Win+Shift+S`
- **Annotation:** [Annotely](https://annotely.com) (free, browser-based) or Figma
- **Resize:** 1200×630px for social cards, 1920×1080 for full screenshots
- **Format:** PNG (lossless — screenshots contain text which JPEG compresses badly)

## Screenshot 2 — The Diff (Code Change)

**What to capture:** The "Changes" tab of MR !2 showing the actual diff.

**URL:** `https://gitlab.com/amareshhebbar/python-gitlab-forktesting/-/merge_requests/2/diffs`

**How to capture:**
1. Click the "Changes" tab
2. Find the `gitlab/_backends/requests_backend.py` diff block
3. Expand it fully
4. Capture the diff showing green `+` lines for the urllib3.Retry addition

**What should be visible:**
```diff
+import urllib3
+
 class RequestsBackend(BackendBase):
     def _build_session(self):
+        retry = urllib3.Retry(
+            total=5,
+            backoff_factor=0.5,
+            status_forcelist=[429, 500, 502, 503],
+        )
+        adapter = requests.adapters.HTTPAdapter(max_retries=retry)
+        session = requests.Session()
+        session.mount("http://", adapter)
+        session.mount("https://", adapter)
+        return session
```

**Annotations to add:**
- Box around the entire green block with label "AI-generated fix — 12 lines"
- Arrow pointing to `status_forcelist=[429, ...]` with label "Fixes the 429 crash"

**Where to use:**
- Devpost "What it does" section
- README before/after proof
- Architecture docs

**Filename:** `screenshots/02_diff.png`
